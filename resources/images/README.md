# IIUM Library Facility Images

This folder contains images for the library facilities.

## Naming Convention:
- Use facility ID as filename
- Format: {FacilityID}.jpg or {FacilityID}.png
- Examples:
  - SR001.jpg (Study Room 001)
  - CR-L1-01.jpg (Carrel Room Level 1-01)
  - DR-01.jpg (Discussion Room 01)

## Image Specifications:
- Recommended size: 800x600 pixels or larger
- Format: JPG, PNG
- Aspect ratio: 4:3 or 16:9 preferred
- File size: Under 2MB per image

## How to Add Images:
1. Take photos of the actual facilities
2. Resize/crop as needed
3. Save with correct facility ID as filename
4. Place in this resources/images/ folder
5. Restart the application to see the images
